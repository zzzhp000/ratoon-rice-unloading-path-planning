# Unloading Path Planning for Ratoon Rice Harvesting

This repository provides a compiled MATLAB application used in the study:

"Optimal Compaction Rate Operational Path Planning for Main Crop Harvest of Ratoon Rice Harvester Based on Grain Unloading Method Constraints"

The executable program together with example datasets allows users to reproduce representative simulation results presented in the paper.

---

## Repository Structure

program/
Compiled MATLAB application installer.

program/program/release/package/
MyAppInstaller.exe

example_data/
field1.xlsx  
Example dataset for Field 1 used in the simulation.

example_results/
Simulation results of Field 1 corresponding to Fig.17 in the paper.

Path planning process video.mp4  
Visualization of the path planning process.

---

## Requirements

MATLAB Runtime (R2022a or later)

Download:
https://www.mathworks.com/products/compiler/matlab-runtime.html

---

## How to Run the Program

1 Install MATLAB Runtime (R2022a or later).  
2 Navigate to the folder:program/program/release/package/
3 Run
4 Launch the installed application.
5 Load the example dataset located in:example_data/field1.xlsx
6 The program will generate path planning results similar to those presented in the paper.

---
## Reproducibility

The example dataset and executable program allow reproduction of representative unloading path planning results shown in Fig.17 of the paper.
